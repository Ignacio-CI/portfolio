# PROJECT SUMMARY

This a personal portfolio website made in Astro and Svelte "island" components. 
The website includes a chat with my digital "Twin" that has perfect knowledge about my skills and professional background.

# Goals

- Make the website feels modern, enterprise and edgy.
- Emphasize the beauty of the frontend application using modern CSS and some 3D effects.

# Code standards and guilines

1. Simpler is better - do not over complicate.
2. Comments only when necessary.
3. Be concise, short README, no emojis.
4. IMPORTANT: Avoid overly defensive programming
5. Write concise, technical code with accurate Astro 5^ syntax.
5. When using Svelte for some interactive components, write concise, technical code with accurate Svelte 5^ (runes) syntax.
6. In the frontend app, use ALWAYS `bun run xxx`. NEVER `npm run xxx`.
7. In the backend app, use ALWAYS `uv run xxx`. NEVER `python3 xxx`.
8. Code, comments and documentation should be ALWAYS written in English.

## UI and Styling

- Use CSS for styling approach.
- Use Svelte's v5 built-in transition and animation features.

You are able to use the Svelte MCP server when implementing interactive components, where you have access to comprehensive Svelte 5 documentation.
Execute MCP calls when creating or editing `.svelte` files.
